# === readme ===

data.nc is Etopo5 (5-min resolution ~ 0.08-deg resolution) bathymetry data set downloaded from 
http://iridl.ldeo.columbia.edu/SOURCES/.NOAA/.NGDC/.ETOPO5/.elev/datafiles.html

Dataset Name: Etopo5
Resolution: 5 Minute

the download website is found from https://ferret.pmel.noaa.gov/static/FAQ/custom_plots/topo_bathy_data.html
