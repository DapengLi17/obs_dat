drifter_annualmeans.nc is downloaded from ftp://ftp.aoml.noaa.gov/phod/pub/lumpkin/drifter_climatology/version_3.06/
data set time period: Feb 1979(see Laurindo et al. 2017 section 2.1.1) to Jul 2020 (version 3.06 includes Jul 2020, see reference web 1)
horizontal resoluion: 0.25 deg

two reference papers:
